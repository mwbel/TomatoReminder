# 不忘

一个自用 macOS 番茄钟，界面参考番茄 ToDo / Focus To-Do 这类移动端效率工具的清爽、任务驱动风格，但不复制原应用素材。

## 功能

- 任务列表：优先级、预计番茄数、完成状态。
- 每日计划：按今天、明天、本周末、已计划、已完成、全部任务过滤，并支持搜索。
- 计划创建：新建任务时可直接归入今天、明天、本周末或稍后计划。
- 计时模式：专注、短休息、长休息。
- 全屏计时：计时卡片右上角可进入专注全屏样式。
- 自动切换：番茄结束后进入短休或长休，休息结束后回到专注。
- 今日统计：今日专注时长、番茄数量。
- 本地保存：任务、设置、统计保存在 `UserDefaults`。
- 本地提醒：番茄或休息结束时播放提示音并发送系统通知。

## 运行

```bash
swift run TomatoReminder
```

## 打包成 `.app`

```bash
chmod +x scripts/build_app.sh
./scripts/build_app.sh
open /Applications/不忘.app
```

第一次运行时，macOS 可能会询问通知权限。
